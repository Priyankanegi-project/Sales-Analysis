# Sales-Analysis
Sales data analysis using Python,Power BI,Sql to identify trends,performance and key insights

## Tools used
-Python(Pandas,Numpy)
-Power Bi
-Jupyter Notebook

## Workflow
1.Collected raw sales data from Kaggle
2.Cleaned and transformed data using Python
3.Performance exploratory data analysis
4.Built interactive dashboard in Power Bi

## Dashboard
-Total Sales
-Monthly Trends
-Sum of total profit
-Sum of Quantity
-used filter
